# PRD: Página de Estética de Luxo

## Objetivo
Definir o que faz uma página de luxo converter e estruturar o projeto para atrair o público-alvo A/B.

## Público-alvo
Mulheres e homens de classe A/B buscando procedimentos de rejuvenescimento.

## Elementos Cruciais
1. **Autoridade:** 
   - CRM do médico visível.
   - Área "Sobre a Especialista".
2. **Prova Social:** 
   - Antes e Depois (com disclaimer ético).
   - Depoimentos.
3. **Tratamentos de Destaque:** 
   - Bioestimuladores
   - Ultraformer MPT
   - Lavieen
4. **Conversão:** 
   - Botão flutuante de WhatsApp.
   - Formulário de consultoria.

## Tech Stack Sugerida
- **Core:** React (Next.js recomendado para SEO)
- **Estilização:** Tailwind CSS
- **Animações/Premium Feel:** Framer Motion
