# Smile Connect - Dr. Rafael Mendes

Landing page de alta performance para o Dr. Rafael Mendes, especialista em Odontologia Estética e Harmonização Facial.

## 🚀 Tecnologias

- **Vite** (Build Tool)
- **React** (Frontend Framework)
- **TypeScript** (Language)
- **Tailwind CSS** (Styling)
- **Framer Motion** (Animations)
- **Lucide React** (Icons)

## 🎨 Conceito Visual

O projeto utiliza um design **Architectural Precision**, com foco em luxo técnico, tipografia imponente e profundidade visual (Z-axis).

- **Geometria:** Bordas Sharp (0px - 2px) para sensação de precisão cirúrgica.
- **Cores:** Prata Obsidian, Branco Estéril e detalhes em Cinza Aço.
- **Animações:** Revelações em camadas e micro-interações fluidas.

## 🛠️ Instalação e Desenvolvimento

1. Clone o repositório
2. Instale as dependências: `npm install`
3. Inicie o servidor: `npm run dev`
